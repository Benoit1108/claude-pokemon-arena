Z1 · gradient subtil

Background par type

Deux radial-gradients très basses opacités (.07 / .10 dark) avec la couleur lineage du type dominant. Pas de bg saturé — l'identité du type reste lisible sans saturer la grille.

Z2 · status badges

5 états distincts

Sweet-spot = success, Preview = info, Outclassed = tertiary, Locked = tertiary + lock overlay, Auth requise = warning + icon « person+lock » (différencie clairement du locked niveau).
