M1 · header

Header mobile minimal

52px avec padding-top: env(safe-area-inset-top). Logo + segmented theme (scale 0.86) + user avatar. Pas de tabs — ils vivent dans la bottom-nav. Pas de stars pill — déplacée vers le menu profil sur mobile.

M2 · bottom-nav

5 entrées · pill gradient

Position fixed (ici absolute dans le frame). Hauteur 64px + env(safe-area-inset-bottom). Icônes Lucide-style stroked 22px. Tap target ≥44px sur toute la zone. Active = icon translate -2px + pill 28×3px gradient gold → fire avec glow.

M3 · entrées

Pokédex / Arena / Trail / Zones / Profil

Mêmes 4 sections que desktop + Profil en remplacement du UserMenu desktop. Le UserMenu desktop contient theme retro, logout, settings — sur mobile c'est la page /profil qui héberge tout ça.

M4 · motion

Économe par défaut

Seule l'animation au changement d'onglet (icon lift + pill apparition) est notable. prefers-reduced-motion coupe le translate. Le backdrop-blur est essentiel — sans lui le contenu défilant sous la bottom-nav casse la lecture.
